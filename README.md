 Customer Churn Analysis and Prediction

 Project Overview

Customer churn is one of the major challenges faced by telecommunication companies. Retaining existing customers is often more cost-effective than acquiring new ones. This project focuses on analyzing customer behavior, identifying factors that contribute to customer churn, and building machine learning models capable of predicting customers who are likely to leave the company.

The project was completed as part of a **Data Science Internship** and follows a structured workflow that includes data preparation, exploratory data analysis, customer segmentation, predictive modeling, and model evaluation.

---

## Objectives

- Analyze customer churn behavior.
- Identify the main factors influencing customer churn.
- Segment customers based on their characteristics.
- Build and compare multiple machine learning models.
- Predict customers at risk of churn.
- Provide business recommendations to improve customer retention.

##  Dataset

The dataset contains customer demographic information, account details, services subscribed, contract information, billing details, and churn status.

Main features include:

- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Phone Service
- Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn

---

#  Project Tasks

##  Task 1 — Data Preparation

### Description

The dataset was loaded and explored to understand its structure. Missing values were handled, categorical variables were encoded into numerical values, and the dataset was prepared for machine learning. Finally, the data was split into training and testing sets.

### Deliverables

- Data Cleaning
- Missing Value Handling
- Feature Encoding
- Train/Test Split


##  Task 2 — Exploratory Data Analysis (EDA)

### Description

Exploratory analysis was performed to understand customer behavior and identify churn patterns.

### Analysis Included

- Overall Churn Rate
- Customer Distribution by Gender
- Partner Status Analysis
- Dependents Analysis
- Tenure Distribution
- Contract Type Analysis
- Payment Method Analysis
- Data Visualization

---

##  Task 3 — Customer Segmentation

### Description

Customers were segmented based on important business features such as tenure, monthly charges, and contract type.

### Objectives

- Identify customer segments
- Analyze churn within each segment
- Detect high-value customers at risk
- Support targeted retention strategies

---

##  Task 4 — Churn Prediction Model

### Description

Several machine learning models were trained and compared to predict customer churn.

### Models Used

- Logistic Regression
- Decision Tree
- Random Forest

### Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score

Hyperparameter tuning and feature selection were also performed to improve model performance.

---

##  Task 5 — Model Evaluation and Interpretation

### Description

The best-performing model was evaluated using the testing dataset.

### Evaluation Included

- Confusion Matrix
- ROC Curve
- AUC Score
- Feature Importance Analysis
- Model Interpretation
